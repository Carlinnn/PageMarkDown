Github

crie um novo repositório na linha de comando

echo "# test" >> README.md
git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin (link do git)
git push -u origin main

ou envie um repositório existente a partir da linha de comando

git remote add origin (Link do git)
git branch -M main
git push -u origin main
